## AI chat sample on GigaChat basis.    

## Frontend stack
- React.js
- Ant Design
- Tanstack/react-query

## Backend stack
- Node.js
- Express
- LowDB

## How to run
  - Please obtain GIGACHAT_AUTH_KEY from GigaChat Studio and edit the "./server/.env" file.

# Start proxy server
$ cd server   
$ npm install   
$ npm run start   
    
# Start frontend client    
$ cd client   
$ npm install   
$ npm run dev
